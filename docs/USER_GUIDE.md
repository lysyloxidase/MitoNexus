# User Guide

## What MitoNexus Does

MitoNexus turns a structured blood panel into:

- marker-by-marker mitochondrial interpretation
- cascade-level impact summaries
- prioritized therapy recommendations
- a 3D knowledge graph
- a 3D mitochondrion overlay
- a downloadable PDF report

## Typical Workflow

1. Open `/analyze`.
2. Enter patient age, sex, and test date.
3. Fill in the available markers and leave unmeasured markers blank.
4. Submit the panel.
5. Wait for the report page to finish processing.
6. Review:
   - overall MitoScore
   - abnormal markers
   - affected cascades
   - therapy shortlist
   - graph view
   - mitochondrion view
   - PDF export

## Understanding the Visualizations

### Knowledge graph

- Red nodes: markers
- Blue nodes: genes
- Green nodes: cascades
- Purple nodes: therapies
- Amber nodes: pathways

### Mitochondrion view

- Warmer colors indicate lower ETC activity
- Cooler or greener regions indicate higher preserved activity
- Tooltips explain which markers are contributing to each ETC estimate

## PDF Report

The PDF includes:

1. cover page and MitoScore summary
2. executive summary
3. marker analysis
4. signaling cascade section
5. ETC status
6. therapy plan
7. monitoring schedule
8. references
9. medical disclaimer

## Important Reminder

MitoNexus is an educational decision-support tool. It does not replace clinical judgment or
qualified medical care.
